# 🎵 Music Store SQL Analysis Project

## 📌 Project Overview

This project focuses on analyzing a music store database using SQL. The database contains information about customers, invoices, employees, artists, albums, tracks, playlists, genres, and media types. The objective is to extract meaningful business insights by writing SQL queries ranging from basic data retrieval to advanced analytical queries involving joins, aggregations, subqueries, and Common Table Expressions (CTEs).

This project demonstrates practical SQL skills used by Data Analysts for solving real-world business problems and making data-driven decisions.

---

## 🎯 Project Objectives

- Analyze customer purchasing behavior.
- Identify top-selling artists, albums, and genres.
- Determine revenue trends and sales performance.
- Understand customer distribution across countries and cities.
- Evaluate employee and customer support performance.
- Practice advanced SQL concepts on a real-world relational database.

## 🗂️ Database Schema

The project consists of the following tables:

| Table Name | Description |
|------------|------------|
| Employee | Employee details and reporting hierarchy |
| Customer | Customer information and assigned support representative |
| Invoice | Customer purchase transactions |
| Invoice_Line | Individual items purchased in each invoice |
| Track | Music track details |
| Album | Album information |
| Artist | Artist information |
| Genre | Music genre categories |
| Media_Type | Audio format information |
| Playlist | Playlist details |
| Playlist_Track | Mapping between playlists and tracks |
